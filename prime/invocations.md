# Agent Invocations

> Copy-paste-ready invocation templates for each agent. Use these to start a session with a specific agent.

## Prime (Orchestrator)
```
Show me the system pulse. What should we work on?
```

## Scout
```
@scout Run a signal scan for [topic/thesis]. Check [sources] for relevant developments in the last [timeframe].
```

## Synthesizer
```
@synthesizer Build a thesis from these signals: [list signals or link to signal files]. What's the structural insight?
```

## Writer
```
@writer Convert thesis [THS-XXX] into a [Substack article | LinkedIn post | conference talk]. Production order: Substack first.
```

## Connector
```
@connector I want to reach [person/organization]. Build an outreach strategy based on [shared context/thesis].
```

## Planner
```
@planner I want to build [idea]. Run the 4-stage pipeline: context → architecture → critique → Build Handoff Spec.
```

## Builder
```
@builder Execute [PLN-XXX]. Follow the Build Protocol: Phase 0 verification → Phase 1-N build → Post-Build validation.
```

## Industry Analyst
```
@industry_analyst Map the industry structure for [sector/company]. Full Step 0-9 methodology.
```

## Investment Analyst
```
@investment_analyst Produce investment-grade analysis for [company/sector] using the structural map from Industry Analyst.
```

## Experimenter
```
@experimenter Design and run an experiment to validate: [specific claim from a thesis]. Define hypothesis, method, success criteria.
```

## Clerk
```
@clerk Check all commitments and deadlines. Flag anything overdue or at risk.
```
