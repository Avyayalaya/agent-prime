# Agent Prime

**A system that builds systems.**

Seven architectural layers, each a markdown file, each compounding on the last. Identity, memory, agents, orchestration, skills, craft, and a recursive loop that gets smarter every session.

A single person operating with the output, consistency, and strategic depth of a small team — with every session compounding on the last.

## The Seven Layers

| Layer | What it does |
|-------|-------------|
| **Identity** | One file that encodes how you think. AI becomes dramatically more effective when it knows your reasoning patterns. |
| **Memory** | Corrections compound across every session. Fix something once, it propagates everywhere. |
| **Agents** | Eleven specialized agents — scout, synthesizer, writer, planner, builder, and more. Each works on its own. |
| **Orchestration** | A registry, dispatch queue, and briefing pipeline. Wake up to a system that already knows the plan. |
| **Skills** | Plug domain expertise into any agent. Encoded methodologies, portable across agents. |
| **Craft** | HTML templates that turn markdown into publication-quality output carrying your voice. |
| **The Loop** | Every session produces learnings. Learnings propagate. The system audits itself and pushes work forward. |
